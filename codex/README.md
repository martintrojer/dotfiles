# Codex Configuration

This package manages Codex skills via GNU Stow.

## Install

```bash
stow codex
```

This links `~/.codex/skills/*` to the skill folders in `claude/mtrojer-plugin/skills/`.

## Included Skills

- `brainstorm`
- `changelog`
- `write-plan`
- `execute-plan`
- `code-reviewer`
- `test-reviewer`
